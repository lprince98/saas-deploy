# The Digital Curator — 서비스 컨텍스트 문서

> 이 문서는 `design/` 폴더 내 6개의 HTML 목업 파일을 분석하여 작성된 서비스 전체 명세서입니다.

---

## 1. 서비스 개요

**The Digital Curator**는 지식 노동자, 연구자, 크리에이티브 전문가를 타깃으로 하는 **프리미엄 노트 테이킹 SaaS** 플랫폼입니다.

- **슬로건**: "당신의 사유를 위한 디지털 갤러리"
- **핵심 가치**: 미니멀리즘, 에디토리얼 미학, 집중력, 의도적인 큐레이션
- **경쟁 포지셔닝**: 단순한 메모 앱(Notion, Apple Notes)에서 탈피하여 _프리미엄 집필 경험_ 과 _AI 기반 지식 조직화_ 를 결합한 제품
- **사용자 규모 언급**: 50,000명 이상의 전문가 사용자

### 핵심 차별점

| 구분 | 설명 |
|------|------|
| 에디토리얼 UI | 잡지·갤러리 수준의 타이포그래피와 여백 중심 레이아웃 |
| AI 문맥 검색 | 단순 키워드가 아닌 의도·맥락 기반 노트 발견 |
| AI 큐레이션 | 노트 자동 태깅 및 아이디어 간 연결 고리 매핑 |
| 글로벌 동기화 | 온/오프라인, 멀티 디바이스(macOS · iOS · Web) |
| 제로 지식 보안 | End-to-End 암호화 기반 개인정보 보호 |
| 버전 스냅샷 | 노트의 과거 시점 복원 기능 |

## 2. 브랜드 & 디자인 시스템

### 2-1. 브랜드 아이덴티티

- **서비스명**: The Digital Curator
- **앱 내 명칭**: "나의 갤러리" / "개인 작업 공간"
- **아이콘**: `auto_awesome` (Material Symbols) — 별·스파클 이미지
- **언어**: 한국어 기본 (다국어 지원 UI 요소 포함)

### 2-2. 색상 팔레트 (Material Design 3 기반)

| 토큰 | 헥스값 | 용도 |
|------|--------|------|
| `primary` | `#24389c` | 주요 CTA, 강조 요소 |
| `primary-container` | `#3f51b5` | 그래디언트 보조 |
| `on-primary` | `#ffffff` | 프라이머리 위 텍스트 |
| `secondary` | `#4a626d` | 본문 보조 텍스트 |
| `secondary-container` | `#cde6f4` | 소프트 배경, 뱃지 |
| `surface` | `#f7f9fc` | 기본 배경 |
| `surface-container-low` | `#f2f4f7` | 카드·패널 배경 |
| `surface-container-lowest` | `#ffffff` | 최상단 컨텐츠 영역 |
| `outline-variant` | `#c5c5d4` | 미세 구분선 |
| `error` | `#ba1a1a` | 에러 표시 |

> **Hero Gradient**: `linear-gradient(135deg, #24389c 0%, #3f51b5 100%)`

### 2-3. 타이포그래피

| 역할 | 폰트 | 위치 |
|------|------|------|
| `headline` | Manrope (700~800) | 제목, 브랜드명, CTA |
| `body` | Inter (400~600) | 본문, 설명 텍스트 |
| `label` | Inter (400~600) | UI 레이블, 버튼 |

### 2-4. 공통 UI 패턴

- **Glassmorphism**: `backdrop-filter: blur(20px)`, 반투명 백그라운드 (`rgba(255,255,255,0.8)`)
- **Editorial Shadow**: `box-shadow: 0px 32px 64px -12px rgba(25,28,30,0.06)`
- **Bento Grid**: 컨텐츠를 크기 다른 카드로 비대칭 배치하는 레이아웃
- **Hover 마이크로 애니메이션**: `hover:translate-y-[-4px]`, `hover:translate-x-1` 등
- **원형 배경 장식**: `blur-3xl` 처리된 대형 원형 색상 블롭

## 3. 페이지별 상세 명세

### 3-1. 랜딩 페이지 (`landing.html`)

**목적**: 서비스 소개 및 신규 가입 유도

#### 구성 섹션

1. **Navigation Bar** (Fixed)
   - 로고 (The Digital Curator)
   - 메뉴: 요금제 / 솔루션 / 리소스
   - 우측 CTA: 로그인 버튼 + "시작하기" 버튼 (primary 색상)

2. **Hero Section**
   - 배지: "New: AI 기반 문맥 검색 기능 출시"
   - 헤드라인: "당신의 사유를 위한 디지털 갤러리."
   - 서브텍스트: 서비스 가치 설명 (~2줄)
   - CTA 버튼: "무료로 시작하기" (Hero Gradient) + "데모 보기" (Ghost)
   - 우측 이미지: 노트 에디터 UI 스크린샷 + 배경 블롭 장식

3. **Bento Features Grid** (3열 그리드, `bg-surface-container-low`)
   - 타이틀: "모든 디테일에 담긴 정교함."
   - `md:col-span-2 row-span-2` — **리치 텍스트 에디토리얼 슈트** (대형)
   - `1×1` — **클라우드 동기화** (아이콘 중심 소형 카드)
   - `1×1` — **스마트 검색** (primary 배경, 강조 카드)
   - `md:col-span-3` — **보편적 접근성** (macOS · iOS 배지 포함 와이드 카드)

4. **Social Proof Section**
   - 타이틀: "전 세계 큐레이터들의 사랑을 받습니다." / "50,000명 이상"
   - 2열 리뷰 카드 (5성 리뷰 × 2개)
   - 리뷰어 프로필: 사진 + 이름 + 직함
   - 이전/다음 페이지네이션 버튼

5. **CTA Section** (Hero Gradient 배경)
   - 헤드라인: "당신의 레거시를 큐레이팅할 준비가 되셨나요?"
   - CTA: "무료로 시작하기" + "상담 신청하기"
   - 14일 무료 트라이얼 문구, 신용카드 불필요 강조

6. **Footer** (6열 그리드)
   - 컬럼: 브랜드 + 제품 + 리소스 + 회사 + 고객 지원
   - SNS 아이콘 (share, mail)
   - 저작권 / 쿠키 설정 / 글로벌 프라이버시

### 3-2. 인증 페이지 (`auth.html`)

**목적**: 로그인 / 회원가입 통합 인증 화면

#### 구성 요소

1. **배경 장식**
   - 좌상단 + 우하단에 초대형 Blur 블롭 (primary-fixed, secondary-container)
   - 우하단·좌하단: 실제 노트 인터페이스 미리보기 이미지 (회전된 카드, `lg:block`, `grayscale` 처리)

2. **브랜드 헤더**
   - `auto_awesome` 아이콘 (그래디언트 배경 아이콘 박스)
   - 서비스명 + "의도적인 생각의 정리." 슬로건

3. **Auth Card** (Glassmorphism)
   - **탭 스위처**: 로그인 / 회원가입 (Pill 형태, 활성 탭 흰색 배경)
   - **소셜 로그인**: Google + Apple (2열 그리드 버튼)
   - **구분선**: "또는 이메일"
   - **입력 폼**:
     - 이메일 주소 (아이콘 + 레이블 + input, `alternate_email` 아이콘)
     - 비밀번호 (아이콘 + 레이블 + input + 토글 가시성 버튼 + "분실하셨나요?" 링크)
   - **제출 버튼**: "갤러리로 계속하기" (Hero Gradient, Full Width)

4. **하단 링크**
   - 이용약관 / 개인정보 처리방침 동의 안내
   - "도움이 필요하신가요?" / 언어 선택 (한국어)

   - "모든 노트 및 아카이브 보기" 링크

### 3-4. 노트 에디터 (`note.html`)

#### 레이아웃

- **상단 TopNavBar** (Landing과 동일한 공유 컴포넌트, 고정)

- **중앙 에디터 캔버스** (메인)
- **우측 인스펙터 패널** (`w-80`)

#### 에디터 캔버스

- **제목 입력**: 대형 투명 input (`text-3.5rem`, `font-extrabold`, 자리 표시자: "제목 없는 노트")
- **서식 도구 모음 (Toolbar)**:
  - 굵게 / 기울임꼴 / 밑줄
  - 글머리 기호 / 번호 매기기
  - 인용구 / 링크 삽입 / 이미지 삽입
  - 각 버튼은 `rounded-lg`, hover 시 `surface-container-high` 배경
- **에디터 본문**:
  - prose 영역: 일반 텍스트 단락
  - 인라인 이미지 삽입 (둥근 모서리)
  - 굵은 텍스트 (`<strong>`)
  - 글머리 기호 리스트 (custom, primary 색상 점)

#### 우측 인스펙터 패널

| 섹션 | 내용 |
|------|------|
| **메타데이터** | 생성일 / 최근 수정 / 단어 수 |
| **분류** | 태그 뱃지 + "태그 추가" 버튼 |
| **액세스 권한** | 공개/비공개 설정 (아이콘 + 설명) |
| **하단 액션** | "공유" 버튼 + "저장" 버튼 (Hero Gradient) |

#### 특수 UI

- **몰입 모드 트리거**: 우하단 고정 FAB (`fullscreen` 아이콘, `bg-inverse-surface`)
- **Focus Mode 스타일**: `backdrop-filter: blur(40px)` CSS 클래스 정의됨

## 4. 사용자 플로우 (User Flow)

```
랜딩 페이지
    ├── [무료로 시작하기] ──→ 인증 페이지 (회원가입 탭)
    ├── [로그인] ──────────→ 인증 페이지 (로그인 탭)
    └── [요금제] ──────────→ 요금제 페이지
                                    └── [프로로 업그레이드] ──→ (외부 결제) ──→ 결제 완료 페이지
                                                                                        └── [대시보드로 이동] ──→ 대시보드
인증 페이지 (로그인 성공)
```

## 7. 구현 시 참고 사항

### 7-1. 라우팅 구조 (예상)

```
/                       → landing.html (랜딩 페이지)
/login                  → auth.html (로그인)

/pricing                → payment.html (요금제)
/payment/success        → payment-done.html (결제 완료)
```

### 7-2. 인증 처리

- Google OAuth + Apple OAuth (소셜 로그인)
- 이메일/비밀번호 (이메일 인증 방식)
- 세션 관리: `@supabase/ssr` 기반 서버 사이드 세션 처리

### 7-3. 다크 모드

- 모든 파일에서 `darkMode: "class"` Tailwind 설정 적용
- 일부 컴포넌트에 `dark:` 접두사 클래스 이미 적용됨 (사이드바, 탑바)

### 7-4. 반응형 레이아웃 브레이크포인트

- `md:` — 태블릿 이상 (768px~): 사이드바 · 다열 그리드 활성화
- `lg:` — 데스크톱 (1024px~): 에디터 인스펙터 패널 표시, 결제 완료 페이지 2열 구성
- 모바일: 단열 레이아웃, 사이드바 숨김

### 7-5. 결제 통합

- 연간/월간 토글 방식 구독 구조

- 연간 결제 시 20% 할인 적용
- 학생 할인 별도 적용 가능 (50%)
- 영수증 PDF 다운로드 기능
